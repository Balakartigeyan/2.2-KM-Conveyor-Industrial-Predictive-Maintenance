# 2.2-KM-Conveyor-Industrial-Predictive-Maintenance

Real-world industrial data science project using 1 month of conveyor monitoring data.

## Project Objective
Build anomaly detection and predictive maintenance insights for a mining conveyor.

## Sample Analysis Output
![Anomaly Detection](eda_plot.png)

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Key Findings
- Identified abnormal vibration spikes
- Temperature drift correlated with load changes
- Data sufficient for anomaly detection, not yet for failure prediction
